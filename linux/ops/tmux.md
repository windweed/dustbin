# tmux

`$ sudo yum install -y tmux`  

输入tmux即可进入新建的会话中。  

`<C+b>, %`  垂直分割，`<C + b>, "` 水平分割。  
`<C + b>, ↑↓←→`，切换窗格。

`exit` 退出窗格

`<C + b>, d` 分离会话。(可以理解为放到后台)

重新连接时，可以在控制台中键入`tmux attach`来附加窗口，需要先查看有哪些正在运行的会话： `tmux ls`

创建多个窗口 `<C + b>, c`  
切换窗口  
`<C + b>, npw` n下一个，p上一个，w交互

`<C + b>, ,`重命名窗口。

`<C + b>, &` 退出窗口。
